Clickme

Interactive R plots

Clickme is an R package that lets you create interactive visualizations the way you are used to creating static plots in R. Call the clickme function and specify a template (only points or now), some data, and any number of parameters and a visualization will open in your browser in all its interactive goodness.

Example

Install

install.packages("devtools")
library(devtools)

install_github("clickme", "nachocab")
library(clickme)

Use

# simple
clickme(points, 1:10)

# fancy
n <- 500
clickme(points,
    x = rbeta(n, 1, 10), y = rbeta(n, 1, 10),
    names = sample(letters, n, r = T),
    color_groups = sample(LETTERS[1:3], n, r = T),
    title = "Zoom Search Hover Click")
